# E-Voting System

## Features
✅ User Registration  
✅ Election Creation  
✅ Vote Casting  
✅ Real-time Results  

## Tech Stack
- **Backend**: Spring Boot, JPA  
- **Database**: MySQL  
- **API Testing**: Postman  

## How to Run
1. Clone the repo.
2. Set up MySQL in `application.yml`.
3. Run `mvn spring-boot:run`.
